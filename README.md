# sdk-node
Temporal NodeJS SDK

### Development
#### Environment set up
* Clone [sdk-core](https://github.com/temporalio/sdk-core) as a sibling directory of this repository root
```
npm install
npm run compile-proto
```

#### Building
```
npm run clean
npm run build
```
-- OR --
```
npm run clean
npm run build.watch
```

#### Testing
```
npm run test
```
-- OR --
```
npm run test.watch
```
